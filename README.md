# Modelo de Madurez · Gestión de Inventarios

**Kearney – Claro Colombia · 2025**

Herramienta de diagnóstico interactivo para evaluar el nivel de madurez de la gestión de inventarios en operaciones Telco Retail B2B/B2C.

---

## Stack

- **React 18** + **Vite**
- **Recharts** — radar y bar charts
- **Outfit** — tipografía (Google Fonts)

## Estructura

```
inventarios-madurez/
├── index.html
├── vite.config.js
├── package.json
├── public/
│   └── favicon.svg
└── src/
    ├── main.jsx
    └── InventariosMadurez.jsx   ← componente principal
```

## Cómo correr localmente

```bash
# 1. Instalar dependencias
npm install

# 2. Correr en desarrollo
npm run dev

# 3. Build para producción
npm run build
```
